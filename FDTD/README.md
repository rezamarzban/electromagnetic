After running this MATLAB code run `mean(Rin)` command to getting radiation resistance, It will return `75` Ohms approximately which is true, The FDTD use Maxwell equations with numerical method to solving electromagnetic waves problems which is applicable at any antenna length and wave frequency.

According to equations in **Kraus 1988**:227 (Kraus, John D. (1988). Antennas (2nd ed.). Tata McGraw-Hill) and **Balanis 2005**:216 (Balanis, Constantine A. (2005). Antenna Theory: Analysis and Design, 3rd Ed. John Wiley and Sons), And according to these antennas specifications in the real world, The radiation resistance of halfwave dipole antennas is `73` Ohms.
