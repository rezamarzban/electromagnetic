Wait's formula:
![image1](Wait_formula.jpg)

Very complex Wait's formula is derived from Norton-Sommerfeld equations, To calculating this confusing formula by `Symbolic` package at `Octave` use `Wait_formula.m` file. Also be noticed that `SymPy` is lacking to solve and calculating complex integrals with imaginary part! while `Symbolic` package at `Octave` which use `SymPy` doesn't lack.

It calculate normalized radiation resistance (=Real{4蟺鈭哯/畏}) of a sample antenna which is related only to ground wave propagation part (see below image for more info), In few lines of codes!
